A cloud based ‘Software as a Service’ (SaaS) solution to improve processes of business

CubiVue is a ‘Software as a Service’ (SaaS) solution, designed to improve business processes in many industries. This cloud based setup can be use in just couple of hours. The solution is designed to ensure higher and optimized efficiency in order handling, reducing logistic errors, and can produce more efficient performance. I led the mobile app design system across all the industries and optimized the exisitng user experience of the app.

    PROBLEM

    The intial app was serving for a single warehouse industry but later on we expanded support for more major industries which required us to create a unified design system for the app which was called as Cubivue App and it will support all the basic features which will required by all major industries. There was no generalized app for the future industries where our UX resources can map new industries on the exisiting platform to create wireframes and mockups for the customers

    • SOLUTION
    • I built a simplified design system for the mobile application to be used by UX resources for their customers and for the app developers that can utilized this unified design components and features for the base app. The end product helps digitize processes of understanding and creating a standard for the app which can be easily demostratable for the user and for the developers to create base app.
    • APPROACH
    • I worked in a team with a fellow designer and a product manager to design the end-to-end UI/UX for the Android and iOS application which were used for designing the base app and mapping future industries to serve optmized experience for their specified processess. I worked closely with my team to gather design requirements, feedback and conduct user testing on wireframes, functional prototypes, and visual designs, both for Android and iOS. I also worked with a team of developers to ensure delivery of the final product.

    WHO IS THIS FOR?

    User Research was critical to this project since we were designing for different type of industries which were unknown to us. One of the first things we did was draft personas to understand the common key values. We identified various common variables which we can serve to our clients in the base app. We gathered information from different industries where we were already serving our customers.

    FINDINGS

    We identified the common variables we required for the base Cubivue app. By researching and consulting with the stakeholders we streamlined our basic features which we will give in our base app.
